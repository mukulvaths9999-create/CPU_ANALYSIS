# Trend Analysis and Professional Benchmarking of Semiconductor Chips

## Objective
Analyze development trends in CPUs and GPUs, validate against Moore's Law, and prepare datasets for predictive modeling.

## Dataset Overview
Historical data of semiconductor chips, focusing on:
* **Process Size** (nm)
* **Transistor Counts**
* **Energy Efficiency**

## Methodology
* **Data Cleaning:** Robust handling of missing values and ensuring data integrity.
* **Feature Engineering:** Deriving new metrics to better represent performance shifts over time.

## Project Structure
* `CPU_ANALYSIS.ipynb`: Main notebook containing the analysis, visualizations, and modeling.
* `data/`: (If applicable) Folder containing raw and processed datasets.

## Dependencies
* Python 3.x
* pandas, numpy, matplotlib, seaborn, scikit-learn
